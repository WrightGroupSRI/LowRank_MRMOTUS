This folder contains data that accompanies the code at https://github.com/nrfhuttinga/LowRank_MRMOTUS 


The data can be downloaded from https://surfdrive.surf.nl/files/index.php/s/QdOryuR8GKVSzao or by running the following lines in the terminal 

```
cd LowRank_MRMOTUS
bash ./download_data.sh
```
